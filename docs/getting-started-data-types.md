Una uses all the basic JavaScript data types.

## Number

#### Integer

```
1
-1
```

#### Float

```
1.5
-1.5
```

## String

You can declare a string using single or double quotes. 

```
'Alice'
"Bob"
```

## Boolean

```
true
false
```

## JavaScript null and undefined

```
null
undefined
```
