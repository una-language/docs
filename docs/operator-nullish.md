## Nullish coalesing

Symbol: `??`. <br/>
Translated to JavaScript [nullish coalesing operator](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Nullish_coalescing_operator). <br/>
Usually takes 2+ parameters.

```
?? null 2
```
